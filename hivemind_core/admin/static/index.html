Binary Protocol Not Enabled

Enable the Audio Binary Protocol to use voice I/O features with satellites.

What is Binary Protocol?
The binary protocol handles audio data from HiveMind satellites. It's required for:

  • Voice satellite audio streaming
  • Microphone satellite audio capture
  • Server-side STT/TTS processing
